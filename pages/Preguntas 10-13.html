¿Qué son y para qué sirven las normas internacionales?


Las normas internacionales son reglas y directrices establecidas por organizaciones globales para asegurar que productos, servicios y procesos sean seguros, eficientes y consistentes a nivel mundial. Estas normas son vitales para promover la interoperabilidad y el comercio internacional, garantizando que los sistemas funcionen de manera efectiva en diferentes países y regiones.



Ejemplo de una normativa aplicada a redes informáticas:

  • Norma IEEE 802.3: Esta norma, establecida por el Instituto de Ingenieros Eléctricos y Electrónicos (IEEE), define los estándares para Ethernet, que es la tecnología más común para redes locales (LAN). IEEE 802.3 especifica cómo deben ser los cables, las velocidades de transmisión (como 10 Mbps, 100 Mbps, 1 Gbps), y el formato de las tramas de datos que se utilizan para la comunicación en la red.

Diferentes topologías, categorías y modelos de red


Topologías de Red:

  • Topología de Bus: Todos los dispositivos están conectados a un solo cable compartido. Es una estructura sencilla y económica, pero si el cable falla, toda la red se cae.
  • Topología de Estrella: Los dispositivos están conectados a un nodo central, como un switch o un hub. Esta topología es fácil de administrar y permite que si un dispositivo falla, no afecte a los demás.
  • Topología de Anillo: Los dispositivos están conectados en un círculo cerrado. Los datos viajan en una sola dirección, y cada dispositivo actúa como un repetidor, pasando los datos al siguiente dispositivo. Un fallo en un dispositivo o en el cable puede afectar a toda la red.
  • Topología de Malla: Cada dispositivo está conectado a varios otros dispositivos. Esto ofrece una gran redundancia y confiabilidad, pero es costoso y complejo de implementar.
  • Topología de Árbol: Es una combinación de las topologías de estrella y bus, donde los dispositivos están organizados en una estructura jerárquica.

Categorías de Red:

  • LAN (Local Area Network): Una red que cubre un área geográfica pequeña, como una oficina o un edificio. Se utiliza para conectar computadoras y otros dispositivos dentro de una ubicación específica.
  • MAN (Metropolitan Area Network): Una red que abarca una ciudad o una gran área metropolitana. Es más grande que una LAN, pero más pequeña que una WAN.
  • WAN (Wide Area Network): Una red que cubre un área geográfica extensa, como un país o incluso varios países. Internet es el ejemplo más conocido de una WAN.

Modelos de Red:

  • Modelo OSI (Open Systems Interconnection): Este es un modelo de referencia que describe las funciones de un sistema de red en siete capas (física, enlace de datos, red, transporte, sesión, presentación y aplicación). Cada capa tiene responsabilidades específicas y se comunica con las capas adyacentes.
  • Modelo TCP/IP: Este modelo es la base para la comunicación en Internet y está organizado en cuatro capas: capa de enlace, capa de Internet, capa de transporte y capa de aplicación. TCP/IP es más práctico y ampliamente utilizado en comparación con el modelo OSI.


Normas de Cableado Estructurado


El cableado estructurado se refiere a un sistema estandarizado de cables y hardware que proporciona una infraestructura de telecomunicaciones integral en un edificio o campus, permitiendo la transmisión de datos, voz y video. Este sistema es esencial para asegurar una red eficiente y fácil de gestionar.


Normas de Cableado Estructurado:

  • ANSI/TIA-568: Esta es una norma para el cableado de telecomunicaciones en edificios comerciales. Define los requisitos para la instalación de cables y componentes de red, asegurando que puedan soportar las aplicaciones de telecomunicaciones actuales y futuras.
    • Categoría 5e (Cat 5e): Este tipo de cable soporta velocidades de hasta 1 Gbps y se utiliza comúnmente en redes Ethernet de hasta 1000BASE-T.
    • Categoría 6 (Cat 6): Soporta velocidades de hasta 10 Gbps en distancias cortas (hasta 55 metros). Es ideal para aplicaciones de alta velocidad como 10GBASE-T.
    • Categoría 6a (Cat 6a): Es una versión mejorada de Cat 6 que reduce la interferencia y soporta velocidades de 10 Gbps hasta 100 metros.
    • Categoría 7 (Cat 7): Diseñado para soportar 10 Gbps a 600 MHz, con blindaje adicional para reducir la interferencia, es adecuado para entornos con alta interferencia electromagnética.
